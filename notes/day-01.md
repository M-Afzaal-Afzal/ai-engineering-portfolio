# Day 1 Notes - Repo and FastAPI Setup

## 5 things I learned

1. FastAPI is a Python framework for building APIs, similar to Express.js in Node.js.
2. `uv` is a modern Python tool for managing dependencies and running commands.
3. Python projects commonly use `pyproject.toml`, similar to how Node.js uses `package.json`.
4. `.venv` is like an isolated Python environment for one project.
5. A `/health` endpoint is used to check if the backend is running correctly.
